## 🎯 About
 
Multiplication Table JS is a lightweight browser tool that generates a multiplication table on demand. The user inputs a number and an upper multiplier limit, submits the form, and the table is dynamically rendered on the page — no page reload required. Built to practice DOM manipulation and event handling with vanilla JavaScript.
 
## ✨ Features
 
- 🔢 **Custom Number Input** - Generate the table for any number you choose
- 🔁 **Adjustable Range** - Set the multiplier limit (defaults to 10)
- ⚡ **Instant Rendering** - Table is built dynamically via DOM manipulation, no reload needed
- 🎨 **Clean UI** - Gradient background with a centered card layout
 
## 🛠️ Built With
 
- **HTML5** - Semantic form structure with labeled inputs and a results container
- **CSS3** - Flexbox layout, gradient background, and card-style components
- **JavaScript** - DOM selection, form event handling, `DOMParser`, and dynamic element creation
 
## 🚀 Getting Started
 
### Prerequisites
 
- A modern web browser (Chrome, Firefox, Safari, Edge)
 
### Installation
 
```bash
# Clone the repository
git clone https://github.com/Luan-Neumann-Dev/multiplication-table-js.git
 
# Navigate to project directory
cd multiplication-table-js
 
# Open in browser
open index.html
```
 
## 📁 Project Structure
 
```
multiplication-table-js/
├── index.html          # Main HTML file with form and results container
├── css/
│   └── style.css       # Styles — gradient background, card layout, table rows
└── js/

```
 
## 🎓 What I Learned
 
- Handling form `submit` events and preventing default browser behavior
- Dynamically creating and injecting HTML elements into the DOM
- Using `DOMParser` to safely parse HTML template strings
- Structuring JavaScript with clear separation between element selection, functions, and events
